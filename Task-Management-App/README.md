# 📋 Task Management App

A modern and responsive Task Management Web Application built using **React**, **Firebase**, and **Tailwind CSS**. It helps users organize tasks, track progress, and stay productive with reminders and an intuitive dashboard.

---

## 🚀 Features

- 🔐 Firebase Authentication (Sign Up / Login / Logout)
- ➕ Add, Edit, Delete Tasks
- ✅ Mark Tasks as Completed
- 📅 Due Date Management
- ⚡ Priority Levels (High, Medium, Low)
- 📂 Category Filters
- 🔍 Search Tasks
- 📊 Dashboard Statistics
- 📈 Task Progress Bar
- 🔔 Today's Reminder Section
- ⚠️ Overdue Tasks (Needs Attention)
- 🎉 Confetti Celebration on 100% Task Completion
- 🌙 Dark Mode
- 📱 Fully Responsive Design
- 🍞 Toast Notifications
- 🗑️ SweetAlert2 Delete Confirmation
- 🔔 Browser Notifications

---

## 🛠️ Tech Stack

- React.js
- Firebase Authentication
- Cloud Firestore
- React Router DOM
- Tailwind CSS
- React Hot Toast
- SweetAlert2
- Canvas Confetti
- Vite

## 📦 Installation

Clone the repository:

```bash
git clone <your-github-repository-link>
```

Go to the project folder:

```bash
cd Task-Management-App
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
